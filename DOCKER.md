# Docker
1. [Install Docker](https://www.docker.com)
1. [Start Docker](#start-docker)
1. [Stop Docker](#stop-docker)

### Start Docker
    ```
    # open a command window
    # from the adopt-a-drain/ folder
    
    docker-compose build
    
    docker-compose up
    ```
    
### Stop Docker
    ```
    # open a command window
    # from the adopt-a-drain/ folder
    
    docker-compose down
